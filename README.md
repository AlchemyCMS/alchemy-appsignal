# Alchemy::Appsignal
[AppSignal](https://www.appsignal.com) error reporting extension for AlchemyCMS.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'alchemy-appsignal'
```

And then execute:
```bash
$ bundle
```
